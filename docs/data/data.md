# Data

RetainAI uses the IBM HR Analytics Employee Attrition dataset as its initial public benchmark dataset.

The dataset is used to build a reproducible employee retention analytics pipeline covering:

- dataset acquisition;
- schema validation;
- preprocessing;
- train / validation / test generation;
- downstream modeling and dashboard analytics.

The dataset is not redistributed in this repository. Users must download it directly from Kaggle and comply with the dataset terms.
