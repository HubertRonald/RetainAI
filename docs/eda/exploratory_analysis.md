# Exploratory Data Analysis

The exploratory analysis layer studies employee attrition patterns across demographic, job-related, compensation and satisfaction variables.

Core analytical areas:

- attrition distribution;
- attrition by department;
- attrition by job role;
- attrition by overtime;
- attrition by compensation bands;
- numerical correlation analysis;
- candidate attrition drivers.

EDA notebooks are declarative. Reusable logic is implemented in `modules/eda/`.
